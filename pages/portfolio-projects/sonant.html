Sonant

Intelligent eBook Reader with TTS

In Development

Overview

Sonant is a cross-platform eBook reader that brings your books to life with high-quality text-to-speech powered by AWS Polly. Whether you're commuting, exercising, or simply prefer listening to reading, Sonant transforms any EPUB or PDF into an immersive audio experience.

The application supports multiple eBook formats with intelligent text extraction and chapter navigation. AWS Polly's neural voices provide natural-sounding narration in multiple languages, making it perfect for both native content and foreign language learning.

Built with Flutter for true cross-platform support, Sonant runs on iOS, Android, macOS, Windows, Linux, and web browsers. Firebase handles user authentication and cloud storage for syncing your library and reading progress across all devices.

Approach & Architecture

Sonant leverages Flutter's cross-platform capabilities to maintain a single codebase for all target platforms. The app uses just_audio for seamless audio playback with support for background audio, lock screen controls, and notification media controls on mobile platforms.

Document parsing is handled by specialized libraries: epubx for EPUB files and Syncfusion's Flutter PDF library for PDF documents. Text is extracted chapter by chapter and processed in chunks to optimize AWS Polly API calls and caching.

Firebase Cloud Functions serve as the bridge to AWS Polly, handling authentication, rate limiting, and audio file caching. Generated audio is stored in Firebase Storage for quick retrieval on subsequent listens, reducing costs and improving response times.
