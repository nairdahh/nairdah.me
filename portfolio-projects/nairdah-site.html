nairdah.me

Personal Portfolio & Virtual Photography Showcase

Active

Overview

nairdah.me is my personal corner of the internet - a Jekyll-powered static site that serves as both a portfolio and a creative outlet. Built on the portfolYou theme with extensive customizations, it showcases my development projects alongside my passion for virtual photography.

The site features a dedicated virtual photography gallery where I share screenshots captured in various games using ReShade, photo modes, and visual modification tools. Each image represents hours of careful composition, lighting adjustment, and post-processing to create compelling visual narratives within game worlds.

A blog section houses my thoughts on games, photography techniques, and occasional tutorials on achieving specific visual effects in different game engines. The entire site is optimized for SEO and performance, with automatic sitemap generation and social media meta tags.

Approach & Customization

Starting from the portfolYou Jekyll theme, I extensively customized the design to reflect my personal aesthetic. The site uses SCSS for maintainable stylesheets with variables for colors, typography, and spacing that can be easily adjusted site-wide.

Liquid templates power the dynamic content generation, including the project cards grid, blog post listings, and photography galleries. Jekyll collections are used to organize different content types (projects, posts, photos) with custom front matter for metadata.

The virtual photography section was a custom addition, featuring a masonry-style gallery layout with lightbox functionality. Each image includes EXIF-like metadata showing the game, tools used (ReShade presets, camera mods), and any post-processing applied.

Performance optimization was key - images are served in optimized formats, CSS is minified in production, and the static nature of Jekyll means lightning-fast page loads with no server-side processing required.
